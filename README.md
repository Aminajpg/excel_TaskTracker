# excel_TaskTracker

A simple task tracking and dashboard project built in Excel while learning core Excel formulas and basic data analysis.

## Project Overview

This project tracks tasks and summarizes them in a small dashboard to monitor progress and workload.

The main focus of the project was practicing Excel logic and formulas such as counting tasks by status and identifying overdue work.

## Key Skills Practiced

* COUNTIF
* IF statements
* TODAY function
* Basic dashboard creation
* Data organization in Excel

## Improvements I Added

Compared to the original tutorial project, I made some modifications:

* Added a **Priority column** to classify tasks
* Slightly redesigned the **dashboard layout**
* Adjusted formulas to fit the new structure

## Learning Purpose

The goal of this project was to practice:

* Working with structured datasets
* Using formulas to summarize information
* Building a simple dashboard in Excel

## Note

The project was inspired by a YouTube tutorial.
I recreated the dataset and adapted parts of the dashboard to practice my Excel skills.
